## Adding Timesheet
- Navigate to Tasks
- Select any completed task
- Now click on Add Timesheet
:::note
You can only add timesheet to the completed task. You can mark any completed by clicking on Mark Complete or Mark Complete Recreate. The later option will also recreated the task based on the set perodicity. 
For more details [view this guide](/guides/3-tasks)
:::
- You can view the added timesheet entry by first selecting the task and then clicking on View Timesheet.
:::note
View Timesheet button will only appear if there is any associcated timesheet entry.
:::
